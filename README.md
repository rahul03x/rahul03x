# 💼 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built using **HTML5, CSS3, and JavaScript**. This project showcases my skills, education, experience, and contact details in a clean and professional layout.

---

## ✨ Features

- Fully responsive design for all devices
- Animated typing effect using Typed.js
- Smooth scrolling navigation
- Fixed header with mobile-friendly menu toggle
- About section with personal introduction
- Skills section (Commerce, Programming, Data & Full Stack)
- Education timeline (School & College)
- Experience section 
- Contact form 
- Social media integration (LinkedIn, GitHub, Instagram)
- Interactive hover animations and transitions

---

## 🛠️ Technologies Used

- **HTML5** 
- **CSS3** 
- **JavaScript**
- **Typed.js** 
- **Boxicons** 
- **Visual Studio Code (VS Code)**
